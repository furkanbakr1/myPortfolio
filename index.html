Merhaba, Ben Furkan Bakır

Frontend Developer olarak modern web teknolojileri ile çalışmaktan keyif alıyorum.

Teknoloji dünyasındaki hızlı değişimleri yakından takip ediyor, kendimi sürekli geliştiriyor ve öğrendiklerimi projelerime yansıtıyorum. Clean code prensiplerine uygun, sürdürülebilir ve performanslı uygulamalar geliştirmeye özen gösteriyorum.
